## 一 介绍
#### 1.1 概述
对于飞鼠NAT组网的非中心控制器部署模式做出群晖的部署教程。设备支持涵盖支持docker的群晖NAS
#### 1.2 支持硬件
支持各大主流平台，详细测试如下。
| 系统 | 典型配置 | 是否支持飞鼠 | 测试问题 |
|:-----:|:-----:|:-----|:-----|
| 7.2 | ARM64 | 是 | 无 |
| 7.2 | X86_64 | 是 | 无 |
| 6.1/2 | ARM64/X86_64 | 否 | 系统内核缺少驱动 |

#### 1.3支持系统
需系统DSM7及以后群晖NAS部署，同时支持ARM64与X86_64架构。
